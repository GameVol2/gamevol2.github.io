Sweet Lines - Support Page

Your guide to solving issues, requesting new features, and contacting the developers.

Game Overview

Sweet Lines is a fun and engaging puzzle game where players connect pieces in lines of five or more to clear the board. Available on both Android and iOS, the game offers classic mechanics with modern visuals.

General Support Information

If you encounter any issues or have suggestions, the developers are eager to hear from you. Below is a guide on how to report bugs, request new features, or seek help with technical difficulties.

Common Issues

  • Game Crashes: Ensure your device is running the latest OS version. Reinstalling the app might also help.
  • Gameplay Issues: If something in the game isn't working correctly, describe the steps leading to the problem.
  • Feature Requests: Suggest new themes, features, or improvements to make the game more enjoyable.

Contact the Developer

To contact the developers of Sweet Lines, send an email to the address below. Include as much detail as possible:

  • Your device model and operating system version
  • The version of the game you're playing
  • A detailed description of the issue or feature request
  • Screenshots (if applicable)
